## key bindings

### for vim

| Key binding | Action                              |
| :---------- | :---------------------------------- |
| `<c-e>,`    | to trigger **emmet**                |
| `<tab>`     | to trigger auto completion          |
| `gd`        | to open a target documentation      |
| `<leader>h` | to see a target highlight group     |
| `<leader>t` | to open/close **NERDTree**          |
| `<leader>d` | to change current working directory |
